### 数据流

1. **用户上传图片**
   - 前端: 用户选择图片 → 预览 → 调节参数
   - 前端: 点击"生成"按钮

2. **前端发送请求**
   - 构建 FormData (图片 + 参数)
   - POST 请求到 /api/transfer
   - 显示加载状态

3. **后端处理**
   - 接收图片和参数
   - 加载 Stable Diffusion 模型
   - 执行 img2img 推理
   - 保存结果图片
   - 返回结果 URL

4. **前端展示结果**
   - 接收结果 URL
   - 显示生成的图片
   - 提供下载选项

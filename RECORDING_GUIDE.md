### 录屏技巧

1. **语速**
   - 保持正常语速，不要太快
   - 给观众时间理解
   - 重要部分可以重复

2. **镜头移动**
   - 平稳移动，避免快速跳转
   - 突出重点区域
   - 给每个部分足够的展示时间

3. **音频质量**
   - 使用外接麦克风
   - 避免背景噪音
   - 音量适中，清晰可听

4. **视觉效果**
   - 使用高对比度的主题
   - 放大字体便于观看
   - 突出重点内容

### 视频编辑 (可选)

如果需要编辑视频：

1. **添加字幕**
   - 使用 Subtitle Edit 或 Aegisub
   - 标注关键概念

2. **添加标题**
   - 项目名称
   - 学号和姓名
   - 关键功能

3. **添加背景音乐**
   - 使用免费音乐库
   - 音量不要过大

4. **导出设置**
   - 格式: MP4
   - 分辨率: 1920x1080
   - 比特率: 5000-8000 kbps
   - 文件大小: 100-200 MB

### 常见问题

**Q: 录屏时卡顿怎么办？**
A: 
- 关闭其他应用
- 降低分辨率
- 使用硬件加速

**Q: 音频不清晰？**
A:
- 检查麦克风设置
- 调整音量
- 使用降噪软件

**Q: 视频文件太大？**
A:
- 降低比特率
- 使用视频压缩工具
- 缩短视频长度

**Q: 如何在视频中显示学号和姓名？**
A:
- 在录屏时展示前端界面（已显示）
- 在视频开头添加字幕
- 在视频结尾显示信息卡
